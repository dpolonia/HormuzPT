## Anomaly check coverage

The new `anomalyCheck` utility validates the following indicators against expected magnitude bands:

- Public debt (M€): expected range [150,000 – 400,000]
- Public debt (% GDP): expected range [50 – 200]
- Nominal GDP quarterly (M€): expected range [30,000 – 120,000]
- GDP growth (%): expected range [-15 – 20]
- Unemployment rate (%): expected range [2 – 25]
- HICP / CPI YoY (%): expected range [-5 – 20]
- Interest rates (%): expected range [0 – 15]

Anomalies are flagged in the API response (`anomalies` field) and logged via `console.warn`. They do not reject data or crash routes.

## Impact

Improves macro data integrity and prevents magnitude/unit mapping errors from reaching the frontend undetected.
